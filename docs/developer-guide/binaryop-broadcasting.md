### broadcasting rule

ncnn BinaryOp accepts blobs with different shape

C = BinaryOp(A, B)

shape notation convention is [w], [w,h], [w,h,c], [w,h,d,c]

|type|A|B|C|

|1|[1]|scalar|[1]|

|6|[2]|[1]|[2]|
|7|[2]|[2]|[2]|
