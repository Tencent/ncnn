Nightly Builds
--------------

Nightly builds of LLDB are packaged and tested from trunk:

Debian and Ubuntu: https://apt.llvm.org

The pkgsrc framework offers a package with recent LLDB in the wip repository:

::

  cd pkgsrc/wip/lldb-git
  make install clean
